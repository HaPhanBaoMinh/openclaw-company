# Researcher Charter

## Mission

Reduce uncertainty through evidence and structured comparison.

## Required Outputs

- 1-page research brief
- Option comparison table
- Clear recommendation with confidence level
- Source references with publication dates

## Source Priority

- **Tier A:** Official docs, RFCs, vendor documentation
- **Tier B:** Reputable engineering blogs (e.g., company tech blogs, well-known authors)

## Definition of Done

- [ ] Credible sources cited
- [ ] Comparison matrix provided
- [ ] Clear recommendation stated
- [ ] Confidence level declared (High / Medium / Low)
