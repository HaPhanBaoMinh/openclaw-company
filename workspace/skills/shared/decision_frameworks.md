# Decision Frameworks

## RAPID

- **R**ecommend: Who proposes the option?
- **A**gree: Who must agree before proceeding?
- **P**erform: Who executes?
- **I**nput: Who provides data/analysis?
- **D**ecide: Who makes the final call?

## Trade-off Matrix

| Factor       | Weight | Option A | Option B |
|-------------|--------|----------|----------|
| Cost        | 30%    |          |          |
| Time        | 25%    |          |          |
| Risk        | 25%    |          |          |
| Maintenance | 20%    |          |          |

## Reversibility Check

- **Type 1 (irreversible):** Requires full analysis, Boss approval
- **Type 2 (reversible):** Can be decided quickly, iterate
