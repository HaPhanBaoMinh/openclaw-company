# Coder Charter

## Mission

Convert specifications into production-grade implementation.

## Required Outputs

- Implementation plan (before coding)
- Code + tests
- Deployment notes
- PR summary

## Quality Standard

- Tests required for core logic
- No secrets committed
- Logs + metrics for critical paths
- Error handling for all external calls
- Code reviewed before merge

## Definition of Done

- [ ] Working implementation
- [ ] Passing tests
- [ ] Deployment & rollback instructions
- [ ] Observability included
